1) Install node.js

2) Install dependencies:

	In the application folder type "npm install"
	
3) Set enviromental variables for .env file(create if not exist):
    PORT=8090
    USERNAME=sa
    SERVER=localhost
    DATABASE=master
    PASSWORD=db_password
    
4) Start server by typing npm start
